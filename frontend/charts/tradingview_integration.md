# TradingView Integration Guide

This document outlines how TradingView charts are integrated into the Wenty.ai trader-coaching UI.

## Overview

TradingView charts are embedded using the TradingView Widget API, which allows for seamless integration of professional charting capabilities into our application.

## Implementation

The integration follows TradingView's standard embedding practices:
- Using TradingView's lightweight charting library
- Embedding charts via iframe or widget API
- Customizing chart appearance to match Wenty.ai's UI theme

## Files

- `simple_chart_example.html` - Basic example of a TradingView chart embed
- `example.js` - JavaScript implementation for dynamic chart loading
